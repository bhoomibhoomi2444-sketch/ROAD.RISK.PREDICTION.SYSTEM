# 🚗 Accident Risk Prediction System

## 📌 Overview

This project is a Machine Learning-based web application that predicts the probability of road accidents based on environmental and weather conditions.

It helps users understand potential risks and provides safety recommendations to reduce accident chances.

---

## 🧠 Features

* 🎯 Real-time accident risk prediction
* 📊 Interactive data analysis dashboard
* 📉 Visualizations (heatmaps, distributions, trends)
* ⚠️ Dynamic risk classification (Low / Moderate / High)
* 💡 Safety recommendations based on conditions
* 🌙 Night & weather-based risk adjustment

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/Anshika-hub-ai/Road_Risk_Analysis_Prediction.git
```

2. Navigate to project:

```
cd Road_Risk_Analysis_Prediction
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```

---

## 📊 Input Parameters

* Visibility (miles)
* Wind Speed (mph)
* Temperature (°F)
* Humidity (%)
* Pressure (in)
* Night Condition (0/1)
* Bad Weather (0/1)

---

## 📈 Output

* Risk Score (%)
* Risk Category (Low / Moderate / High)
* Possible Causes
* Safety Precautions
